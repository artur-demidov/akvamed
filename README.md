### Запуск бэк
- Поднять базу
- php -S localhost:8000 form.php

### Пометки:
- Бэк credentials прям в коде, для тестового ок
- Файл svg спрайтов заинлайнен в html, сам файл отдельно оставил
- Валидацию можно сделать лучше и добавить на клиенте
- Антиспам стоит заменить условия OR и продумать проверку по ip + капчу 
- Работает только форма в модале, для формы в ленте надо продумывать поведение
- Работа с dom элементами сделана через data атрибуты
- Прокрутка лицензий сделана через скролл + нативный js. Не умеет прокручиваться свайпом мышки в сравнении со swiper или slick. На тач устройствах и с тачпадом работает нормально
- Модал сделан через dialog. Можно добавить focus trap и хорошее минималистичное решение получается. Можно использовать готовые micromodal или fancybox
- Для коллапсов применены 2 разные техники анимации, современный interpolate-size: allow-keywords с height: auto и grid-template-rows: 0/1fr
- Элементы немного привел в порядок на свой вкус, убрать хаус высоты строки и размеров
- Футер не сделал
- Шрифты загружаются локально
- Запрос к апи закомментирован и добавлен фейковый ответ со случайным исходом
- Большие картинки пожаты в современные форматы, svg немного дорабатывал в figma и сжимал вручную через svgo
- Код отформатирован через prettier
- Стилистические токены максимально старался выносить в файл variables.css
- Проект раздается как есть, в продакшн конечно лучше использовать сборку и сжатие
- Пути к ассетам странно выглядят для обеспечения работы локально + работы в github pages
